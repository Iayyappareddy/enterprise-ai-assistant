Installation Guide

Enterprise Private AI Assistant runs fully on your local system to ensure maximum privacy and control. Follow the steps below to get set up.

Contents
Prerequisites Installation Steps System Requirements Notes Troubleshooting

Prerequisites

Before installation, ensure the following are installed on your system.

Python
Required for the backend runtime.
↗ python.org/downloads
Docker Desktop
Required for Qdrant and Redis containers.
↗ docker.com/desktop
Ollama
Required for local LLM inference (Llama 3).
↗ ollama.com/download

Installation Steps

01
Download the Package
Download the Enterprise Private AI Assistant package.
02
Extract the Archive
Extract the downloaded ZIP archive to your preferred installation folder.
03
Run the Setup Script
Open the extracted folder and run the setup script. This automatically configures your environment.
setup.bat
Creates virtual environment Installs dependencies Runs DB migrations Pulls Llama 3 model Creates Docker containers
04
Launch the Application
Double-click the launcher. All services start automatically and the UI opens in your browser.
EnterpriseAI.exe
Django backend Celery worker Redis + Qdrant Ollama LLM Application UI

System Requirements

Minimum
Windows 10 / 11
16 GB RAM
SSD storage
Internet (initial setup only)
Recommended
Windows 11
32 GB RAM
NVMe SSD
Multi-core CPU

Notes

First Launch The initial setup may take several minutes — it downloads Llama 3, installs dependencies, and sets up Docker images. Subsequent launches are much faster.
Performance AI response speed depends on available RAM and CPU. 32 GB RAM is recommended for smooth inference.
Offline After Setup After initial setup, the full application runs offline — no internet required for normal use.

Troubleshooting

Docker connection issues
Ensure Docker Desktop is running before launching. Check the system tray icon to verify it is active.
AI model not responding
Verify Ollama is installed correctly. Run ollama run llama3 in a terminal to test it.
Slow AI responses
Close other applications to free memory. Local LLM inference is CPU and RAM intensive — 32 GB is recommended.
setup.bat fails
Ensure Python is on your system PATH. Run python --version to verify. Try running as Administrator.
